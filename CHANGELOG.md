# Changelog

## 0.9.1

- Release candidate for the first public Unity SDK repository.
- Keeps the runtime API focused on create, load, sync, cache helpers, typed sync status, and structured runtime errors.
- Documents `saveId` persistence and integer save `version` semantics through the pinned contract bundle.

## 0.1.0

- Initial Unity SDK candidate for Persistly create, load, sync, cache helpers, typed sync status, and structured runtime errors.
- Includes a sample project and Last Beacon scene for engine validation.
- Pins `persistly-contract-v0.2.0` for OpenAPI, examples, and runtime payload limits.
